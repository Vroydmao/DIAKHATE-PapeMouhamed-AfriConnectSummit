# AfriConnect Summit 2026

**Auteur :** Pape Mouhamed Diakhaté
**Classe :** L1CS

## Description
Site vitrine complet pour une conférence tech panafricaine fictive, développé dans le cadre de l'examen L1 Web.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, variables CSS, media queries, dark mode)
- JavaScript Vanilla (aucun framework)
- Google Fonts (Syne, Plus Jakarta Sans)
- Bootstrap Icons

## Fonctionnalités JavaScript implémentées
- Dark Mode / Light Mode avec persistance localStorage
- Navbar dynamique au scroll + menu hamburger mobile
- Compte à rebours en temps réel
- Compteurs animés au scroll (IntersectionObserver)
- Animations fade-in au scroll
- Onglets du programme (3 jours)
- Filtrage dynamique des intervenants
- Validation complète du formulaire de contact
- Bouton retour en haut
- Année dynamique dans le footer
